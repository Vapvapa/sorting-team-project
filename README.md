# Sorting-team-project
## Описание проекта
Sorting-team-project — приложение для сортировки коллекций кастомных объектов с использованием алгоритма InsertionSort и бинарного поиска.

Цель проекта — разработать приложение, реализующее:
* сортировку коллекций объектов по заданным классам с применением паттерна «Стратегия»;
* бинарный поиск элементов в отсортированных коллекциях;
* многопоточную обработку сортировки (минимум 2 потока);
* различные способы заполнения исходных массивов данных;
* валидацию вводимых данных;
* дополнительные алгоритмы сортировки и функционал записи результатов в файл.

## Используемые классы
В проекте реализованы следующие классы:
* Животное (атрибуты: вид, цвет глаз, наличие шерсти);
* Бочка (атрибуты: объём, хранимый материал, материал изготовления);
* Человек (атрибуты: пол, возраст, фамилия).

Все классы используют паттерн Builder для создания объектов.

## Функционал приложения
### Основной функционал
* выбор способа заполнения исходного массива (из файла, рандом, вручную);
* выбор длины массива;
* сортировка объектов с использованием дженериков;
* реализация бинарного поиска в отсортированной коллекции;
* выполнение сортировки в ThreadPool (многопоточность, минимум 2 потока);
* сортировка по всем трём полям объектов.

### Дополнительные функции
1. Дополнительное задание 1: сортировка объектов по числовому полю с разделением на чётные/нечётные значения.
2. Дополнительное задание 2: запись отсортированных коллекций и найденных значений в файл (режим добавления).
3. Дополнительное задание 3: заполнение коллекций с помощью стримов.
4. Дополнительное задание 4: многопоточный подсчёт количества вхождений элемента N в коллекцию.

## Системные требования
Для работы приложения необходимы:
* Java Development Kit (JDK) версии 19 или выше;
* среда разработки (IDE) с поддержкой Java (например, IntelliJ IDEA, Eclipse);
* доступ к репозиторию на GitHub/GitLab ([https://github.com/Vapvapa/sorting-team-project](https://github.com/Vapvapa/sorting-team-project)).

## Инструкция по запуску
1. Клонируйте репозиторий: ```git clone https://github.com/Vapvapa/sorting-team-project ```
2. Импортируйте проект в IDE.
3. Соберите проект (используйте Maven/Gradle, если они настроены).
4. Запустите основной класс (```Main.java```).

## Управление приложением
После запуска приложения пользователь может:
1. Выбрать тип сущности.
2. Выбрать способ заполнения массива.
3. Задать длину массива.
4. Ввести данные (если выбран ручной ввод).
5. Просмотреть исходные и отсортированные данные.
6. Выполнить бинарный поиск.
7. Выполнить глобальный поиск по всем коллекциям (поиск по числовому значению).
8. Вернуться в главное меню или выйти из приложения.
